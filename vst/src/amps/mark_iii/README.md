# MarkIII

Bundled Mesa-Boogie Mark III R2 / crunch-mode style amp for Rocksmith `Amp_CA85`.

Local references:

- `amps/Mesa Mark III (CA_38)/boogie_mkiii.pdf`
- `amps/Mesa Mark III (CA_38)/mk3-1.gif`
- `amps/Mesa Mark III (CA_38)/mk3-2.gif`

The folder name still says `CA_38`, but the runtime mapping follows the
corrected curation: `Amp_CA85` is the Mark III crunch amp. Rocksmith exposes
only `Gain`, `Bass`, `Mid`, and `Treble`, so `Gain` is voiced as the schematic's
R2 / crunch drive rather than a full lead-channel control.
