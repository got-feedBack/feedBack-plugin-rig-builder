# MarkIV

Bundled Mesa-Boogie Mark IV lead-channel style amp for Rocksmith `Amp_CA38`.

Local reference:

- `amps/Mesa Mark IV (CA_85)/Mesa_Boogie_Mark_IV_Schematics.pdf`

The folder name still says `CA_85`, but the runtime mapping follows the
corrected curation: `Amp_CA38` is the Mark IV amp. Rocksmith exposes only
`Gain`, `Bass`, `Mid`, and `Treble`, so the plugin derives the Mark IV lead
drive, graphic-EQ contour, and presence/feedback behavior from those controls.
