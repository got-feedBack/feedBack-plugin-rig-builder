# Box_2x12_Alnico synthetic IRs

These files are Slopsmith-owned synthetic cabinet impulse responses generated by:

```sh
python3 rig_builder/tools/make_synthetic_alnico_cab_irs.py
```

They model a generic 2x12 open-back alnico guitar cabinet with:

- 48 kHz mono IEEE-float WAV format
- 2048-sample minimum-phase FIRs
- dynamic, condenser, and ribbon mic voicings
- cone, edge, and off-axis positions
- speaker resonance around 75 Hz
- open-back low-mid cancellation
- upper-mid alnico-style chime
- guitar-speaker rolloff above the main band

The generator does not read, analyze, deconvolve, fit, copy, or require any captured
or proprietary IR. It is intended as a redistributable fallback cabinet family for
`Cab_EN212C` and AC30-style tones.
