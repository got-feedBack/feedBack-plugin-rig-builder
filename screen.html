Connect tone3000 account

Sign in to tone3000 to list candidates in-app and unlock automatic downloads — no API key to paste. This opens tone3000 in your browser; you approve once and come back. The plugin only receives a temporary, revocable token.

🚀 Quick start — how this plugin works

Two ordered steps. After you've done them once, the plugin self-heals new songs as you open them.

  1. Scan library & auto-assign — walks every song and assigns captures from the curated 1-to-1 mappings only. Anything uncurated lands in pending for you to pick later. Tick the ⚠ Allow tone3000 fuzzy fallback box below if you want the plugin to auto-search tone3000 for uncurated gears instead.
  2. Refine in the Gear catalog — for each amp set 🎚 Variants (clean / crunch / dist), or assign a NAM/VST via 📚 Library. The Songs editor lets you tweak per-song mic position + gain variant + swap gear.

Scan library & auto-assign captures

Walks every sloppak song in your library and, for each gear in each tone, resolves a NAM / IR / VST in this order: (1) reuse an assignment you've already made for that gear in another song, (2) your curated picks (default_captures.json + gain_variants), (3) a fresh tone3000 search + download — unless you turn that step off below.
